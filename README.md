# Global Billionaire Wealth Analysis (Excel)

## Project Overview
This project analyzes global billionaire data to explore wealth distribution across countries, industries, and demographics. An interactive Excel dashboard was built to summarize key metrics and insights.

## Dataset
The dataset contains information on global billionaires, including net worth, country, industry, gender, age, and self-made status.

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- KPI Metrics

## Key Insights
- Wealth is highly concentrated in a small number of countries
- Technology dominates total net worth by industry
- Most billionaire wealth is self-made

## Conclusion
This project demonstrates Excel-based data analysis and dashboard development skills.
