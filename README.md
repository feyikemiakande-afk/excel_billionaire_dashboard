## Overview
This project analyses global billionaire data to explore how wealth is distributed across countries, industries, and demographic groups. An interactive Excel dashboard was built to support exploratory analysis through dynamic KPIs and user driven filters.

---

## Business Objective
To enable users to compare billionaire wealth distribution across countries, industries, gender, and sources of wealth using clear KPIs and interactive visuals.

---

## Dataset
Global billionaire data including rank, net worth, country, industry, source of wealth, gender, age, amd others.

---

## Tools Used
- Microsoft Excel (data cleaning, Pivot Tables, charts, slicers)

---

## Analysis Performed
- Cleaned and structured raw data
- Aggregated net worth and age metrics
- Calculated total and average net worth and average age
- Ranked individuals for comparative analysis
- Enabled dynamic recalculation using slicers

---

## Core KPIs
- Rank  
- Total Net Worth  
- Average Net Worth  
- Average Age  

---

## Key Business Questions
- How concentrated is billionaire wealth across countries?
- Which industries contribute most to total net worth?
- How does wealth distribution vary by gender and source?
- How do KPIs change when different filters are applied?

---

## How the Dashboard Is Used
- Users apply country, industry, gender, and source slicers
- KPIs and charts update automatically
- Comparisons across regions and demographics become visible

---

## Key Insights & What It Means
**Insights**
- Wealth is concentrated among a small number of individuals and countries
- Technology, Finance, and Retail dominate billionaire wealth
- Male billionaires hold most total net worth
- Wealth accumulation occurs over time

**What It Means**
- Selecting different countries reveals regional concentration differences
- Industry filters show which sectors drive wealth in each country
- Demographic filters highlight shifts in distribution patterns

---

## AS-IS / TO-BE (Analytical View)
- **AS-IS:** Static summaries limit structured comparison  
- **TO-BE:** Interactive filters enable dynamic exploration across variables

## Conclusion
This project demonstrates how Excel can be used to combine strong data analysis with business-focused exploration, enabling users to interact with data and understand how wealth distribution patterns change across key variables.
